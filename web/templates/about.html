About

What is this site? Most aircraft have transponders which are continuously transmitting radio signals with information about the flight. I'm receiving those signals with a USB software defined radio (SDR), saving all of the radio messages into a database, and processing the messages to present the data you see on this site. All of the flight data you see on this flight is what I have been able to receive over-the-air from my house near the Hillsboro Airport (KHIO).

Data sources

The flight data is all logged by the radio receiver at my house.

United States aircraft registration info is from the FAA civil aircraft registration database. Canadian aircraft registration details from from the Transport Canada Civil Aircraft Register Database.

Basic type and registration information for other aircraft is from J. Sun's World Aircraft Database and FlightAware's aircraft data file that is included with dump1090.

Airline data courtesy of openflights.org

Aircraft icons in the flight list are shamelessly pilfered from dump1090, who credits FlightAware as the original source.

Source code

All of the code to log and decode the data, and run this site, is written in Go and available under an open source license on GitHub.
